# DexSpeed Frontend

High-speed Solana DEX for trading EBK (Ebookito) and SPL tokens with minimal fees.

## Quick Start

```bash
npm install
cp .env.example .env
# Edit .env - Set VITE_PROGRAM_ID
npm run dev
```

## Build & Deploy

```bash
npm run build
# Upload dist/ to HostGator public_html/
```

See `/DEPLOYMENT.md` for full deployment guide.
